In OMERO you are able to export any image as an OME-TIFF file, save it as a JPEG, PNG or TIFF or if it was archived at import, download it in the original image format.


  1. Click on the Saving Options icon in the toolbar.



  2. If the image was archived in its original file format select Download from the dropdown menu.



  3. Select Export as OME-TIFF from the dropdown menu to export in that format.



  4. Select Save as ... to save as a JPEG, PNG or TIFF.



  5. Scripts can be used to output in a number of formats.
