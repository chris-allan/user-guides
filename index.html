These pages contain the User Guides for OMERO.insight and OMERO.web.

The Guides for the current versions are listed in the main menu, and the Guides for Previous Versions page has links to archives of older versions.

The Resources page contains links to ZIP archives of the PDFs for all the current versions, customised versions of some of the Guides and other resources such as Omnigraffle® and Word® files.

The current release versions of OMERO.insight are 4.4.8p1 and 5.0.0-beta1. You can tell what version of OMERO.insight you are using by looking at the login screen.

New Features Highlight

Recently the ability to edit channel names was added to OMERO in response to requests from users.

OMERO.insight

To edit the channel names, click on the green Pen button.
Click Save to make the changes to the selected image.
Click Apply to All to save the changes to all images in the dataset.


 

OMERO.web

To edit the channel names, click on the Pen icon.
Click Save to make the changes to the selected image.
Click Apply to All to save the changes to all images in the dataset.
