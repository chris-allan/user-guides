The OMERO User Help materials are created and updated by the OME User Experience Team.

For any issues or question specific to the OMERO User Help web pages or any other OME web pages please use the contact form.

Any questions about the OMERO software, Bio-Formats, source code, bugs, data issues or general OME queries, please use the appropriate area of the Open Microscopy Forums or use the mailing lists.

Using these lists or the forums for all support-related requests means the whole community can take part and benefit. We endeavour to respond to all queries within 2 working days.
